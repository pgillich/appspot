ESP Simulation

by Peter Gillich

Requires

  • Latest Sun/Oracle JDK, perhaps JRE is enough. Tested with JDK 6 Update 24 (1.6.24).
  • Enabled Sun/Oracle Java Plug-in in the web browser. It's enabled by default.
  • Java 3D. After install, browsers must be restarted. Old video cards aren't supported. Tested with Java 3D 1.5.2 (Windows).

Model Description

It's a very simple ESP simulation. See more about ESP on Wikipedia: Electronic Stability Control. See more details about the model in a Mathcad 14 file: ESP.xmcd. The MASON simulation library is used for GUI features.

Usage

Click on Start MASON, select ESP and click on Select. Please wait some seconds for presenting two new windows.

After start, select the Model tab and change some parameter, for example:

  • CorrectingBreakEnabled: Enables/disables correcting break.
  • Dt: Delta t, smaller means slower simulation.
Press the Play button. On the Display window, 3D operations (zoom, pan, rotation) are possible by left/middle/right button + mouse movement.
