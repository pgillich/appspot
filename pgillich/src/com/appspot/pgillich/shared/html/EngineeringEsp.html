ESP

Requires

  • Latest Sun/Oracle JDK, perhaps JRE is enough. Tested with JDK 6 (1.6.24, 1.6.26).
  • Enabled Sun/Oracle Java Plug-in in the web browser. It's enabled by default.
  • Java 3D. After install, browsers must be restarted. Old video cards aren't supported.
    Tested with Java 3D 1.5.2 x86 (Windows XP x86, Windows 7 x86, Windows 7 amd64).
    On an amd64 Windows, with an x86 browser (it's the most common use case), use the x86 installer, instead of Java 3D 1.5.2 amd64 (tested on Windows 7 amd64).

Model Description

It's a very simple 3D kinamatic simulation. See more about ESP on Wikipedia: Electronic Stability Control. See more details about the model in a Mathcad 14 file: ESP.xmcd. The MASON simulation library is used for GUI features.

Usage

If Start MASON appears (please be patient - GAE is a very slow file server), click on it and select ESP and click on Select. Please wait some seconds for presenting two new windows.

After start, select the Model tab and change some parameter, for example:

  • CorrectingBreakEnabled: Enables/disables correcting break.
  • Dt: Delta t, smaller means slower simulation.
Press the Play button. On the Display window, 3D operations (zoom, pan, rotation) are possible by left/middle/right button + mouse movement.
